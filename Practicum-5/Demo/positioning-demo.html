CSS Positioning

Position: static (default)

No offset properties applied
Static Position

Behavior: Element follows normal document flow. Offset properties like top, left have no effect.

Position: relative

Offset: top: 25px; left: 35px
Relative Position

Behavior: Element remains in document flow but is shifted visually. Space is still reserved at its original location.

Position: absolute

Offset: top: 15px; right: 15px
Absolute Position

Behavior: Removed from document flow. Positioned relative to nearest ancestor with position: relative, absolute, or fixed.

Position: fixed

Offset: top: 15px; right: 15px
Fixed Position

Behavior: Anchored to the viewport. Remains visible during scroll. Useful for sticky headers, floating buttons, etc.

Position: sticky

Offset: top: 0
Sticky Position

Scroll down to see sticky behavior.

Behavior: Acts like relative until scroll threshold is met, then sticks to defined offset.

Offset Properties: Default vs. Explicit

Default: When top, left, right, bottom are not specified, they default to auto, meaning no offset is applied.

Explicit: Setting values like top: 20px or left: 30px moves the element based on its positioning context.

  • auto = browser decides placement (usually no movement)
  • Explicit values = override default and reposition element
